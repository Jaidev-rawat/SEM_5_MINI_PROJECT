This project focuses on building a deep learning model which can classify more 
than 40 different classes of Indian food by looking at any unknown image.
i.e. On giving a food image as input it predicts what is the name of the 
food and what is the calorie value associated with it.


RUN PREDICTIONS.ipynb file
In order to run the predictions.ipynb file inside model_training folder:-
Install all the dependency written there.
Follow the instructions inside the file.



In order to run webApp it:-

have all the dependency installed like tensorflow, Pillow, Flask, requests, numpy.
Run the foodtest.py file inside flask web app folder using "python foodtest.py" command in the the terminal.
Go to localHost:8088 and use the webapp as shown in demo video.
